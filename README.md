# Gerador Ticket OpenFinance Frontend

Frontend Angular 19.x inspirado no layout da wiki do Open Finance Brasil.

## Requisitos

- Node.js 20+
- npm 10+

## Instalação

```bash
npm install
```

## Ambiente local

```bash
npm start
```

Aplicação disponível em `http://localhost:4200`.

## Build

```bash
npm run build
```
